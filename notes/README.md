# 2 排序题目

## 2.1 排序模板
排序一般使用 C++ 的sort 函数, 详情看 模板中的排序部分
```c++
//1. 可以全部排序 排序完成之后进行 rank
std::sort(stu, stu + 100010, cmp);  //或者 Student实现 "<"赋值运算符重载
rank(stu , 100010);

//2. 也可以进行部分排序 , 如对 [10, 21) 进行排序, 再rank
std::sort(&stu[10], &stu[21], cmp);
rank(&stu[10], 10);
```
